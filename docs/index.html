How It Works

SZ MUSIC uses a simple yet powerful architecture:

1️⃣ GitHub Repository

Store your audio and video files in a GitHub repository's /uploads folder

2️⃣ Authentication

Users enter a password to unlock upload functionality

3️⃣ Dynamic Loading

The player fetches the file list from GitHub API and displays them

4️⃣ Playback

Click any track to stream it via GitHub's raw content CDN

Code Structure:

const USER = "your-username"; const REPO = "your-repo-name"; const PASS = "your-password"; // Fetch files from GitHub fetch(`https://api.github.com/repos/${USER}/${REPO}/contents/uploads`) .then(r=>r.json()) .then(data=>{ /* display tracks */ }); // Stream media play(name) { audio.src = `https://raw.githubusercontent.com/${USER}/${REPO}/main/uploads/${name}`; }

Frequently Asked Questions

What file formats does SZ MUSIC support?

SZ MUSIC supports MP3 audio files and MP4 video files. These are widely compatible formats that work across all modern browsers without additional plugins.

How do I set up SZ MUSIC for my own repository?

1. Create a GitHub repository, 2. Create an /uploads folder, 3. Copy the HTML code, 4. Replace USER, REPO, and PASS constants with your values, 5. Deploy on GitHub Pages or any web server.

Is the password secure?

The password in the HTML is client-side only. For production use, consider implementing server-side authentication. The current implementation is suitable for basic access control.

Can I upload files directly from the player?

The upload box appears after password authentication. However, uploading to GitHub requires API tokens. For full functionality, you'll need to implement a backend service.

What are the storage limits?

GitHub repositories have no strict file size limits, but individual files should be under 100MB for optimal performance. For very large files, consider using GitHub LFS.

Can I customize the design?

Yes! The HTML uses inline CSS. You can modify colors (like #22c55e), fonts, layout, and add your own branding easily.

Is SZ MUSIC mobile-friendly?

Yes, the entire interface is responsive and works seamlessly on all devices including smartphones, tablets, and desktops.

How do I host this?

You can host SZ MUSIC on GitHub Pages (free), Netlify, Vercel, or any static hosting service. Simply upload the HTML file and it's ready to use.
